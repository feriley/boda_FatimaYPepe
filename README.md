## ⚙️ Instalación

# Frontend
git clone https://github.com/feriley/boda_FatimaYPepe
npm install
npm run dev

# Backend (requiere Docker)
docker build -t boda-api .
docker run -p 8080:8080 boda-api



Variables de entorno:

env
# .env frontend
VITE_API_URL=https://api-web-boda.onrender.com
